# Kids Neck of the Woods Society

This repository is a minimal GitHub Pages site built around the images in [static/](static).

## Main files

- [index.html](index.html)
- [styles.css](styles.css)
- [static/kidsnowsociety.png](static/kidsnowsociety.png)
- [static/secretquestcrate.png](static/secretquestcrate.png)

## Deploy on GitHub Pages

1. Push this repository to GitHub.
2. In repository settings, open Pages.
3. Set the source to the default branch and the root folder.
4. Save and wait for the site to publish.

## Edit the site

- Replace the PNGs in [static/](static)
- Adjust spacing and layout in [styles.css](styles.css)
